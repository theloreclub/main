The Lore Club — minimal scaffold

This workspace contains an initial scaffold for The Lore Club.

Next steps:

- run `npm install`
- set environment variables: `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, `STRIPE_SECRET_KEY`, `CLAUDE_API_KEY`
- run `npm run dev`

This scaffold includes:
- Next.js 14 app router structure
- Tailwind CSS
- basic layout, homepage, and browse page
- `lib/` helpers for Supabase, Stripe, and Claude (stubs)
- simple UI components

I'll continue next with Supabase auth wiring and the listings API.
